# API DevOps - Atividade Formativa

Este projeto consiste em uma API REST básica utilizando **Spring Boot 3** e **Java 17**, desenvolvida para fins de estudo de pipeline de CI/CD e conteinerização.

## Tecnologias
- Java 17
- Maven
- Spring Web

## Como testar
Após rodar a aplicação, acesse: `http://localhost:8080/status`

*Projeto em desenvolvimento para a disciplina de DevOps.*
